# 발주서 → 배대지 주문서 변환기 — 사용설명서

쿠팡 발주서와 로켓장부로 배대지 대량주문서 엑셀을 자동 생성합니다.
배대지 27열 양식·배송정보는 앱에 내장돼 있습니다.

---

## 1. ★ 로켓장부 준비 (가장 중요)

변환 정확도는 장부에 달려 있습니다. 아래 2가지를 해두세요.

### 1-1. "개별수량" 열 추가

쿠팡 발주는 묶음 단위로 와도 중국에선 낱개로 사야 하는 상품이 있습니다.
장부 맨 끝에 **`개별수량`** 열을 만들고 **1발주 안의 낱개 수**를 입력하세요.

| SKU ID | 상품명 | 단가 | **개별수량** |
|---|---|---|---|
| 74011693 | 커튼봉 브라켓 6개 | 2.22 | **6** |
| 74011695 | 집게 고리 10개 | 6.6 | **10** |
| 74012211 | 알람시계 1개 (단품) | 12.51 | **(빈칸)** |

- 단품은 비워두면 자동으로 1 처리
- 열 이름: `개별수량` 권장 (`묶음수량·낱개수량·입수·P수` 도 인식)

> ⚠️ **단가는 "세트(1발주분) 가격" 그대로 둡니다(변경 금지).**
> 개별수량은 수량 칸에만 반영되고 금액(합계)은 바뀌지 않습니다.
> 예) 브라켓 6개: 단가 2.22 그대로 → 수량 12, 합계 4.44

### 1-2. 빈 칸 채우기

단가·상품 URL이 비어 있는 품목은 합계가 빈 채로 출력됩니다.
완전히 나오게 하려면 해당 행의 **단가(위안)·상품 URL·색상·이미지 URL**을 채우세요.
(장부에 SKU 자체가 없으면 행을 새로 추가)

---

## 2. 변환

1. **STEP 01** — 쿠팡 발주서(csv) 업로드
2. **STEP 02** — 로켓장부(xlsx) 업로드
3. **[변환 실행 →]** → **[결과 엑셀 다운로드]**

배송정보·쇼핑몰 URL·양식은 자동 적용됩니다.

---

## 3. 결과 보기

| 항목 | 뜻 |
|---|---|
| 전체 출력 품목 | 주문서에 출력된 전체 |
| 정보 완비 | 단가·URL 있어 합계까지 채워진 품목 |
| 보완 필요 | 장부 빈 칸 때문에 일부가 빈 채로 나온 품목 |
| 위안 합계 | 정보 완비 품목 총액 |

- **모든 품목이 출력**됩니다. 정보가 빈 품목은 그 칸만 빈 채로 나옵니다.
- "보완 필요 리포트" = 장부의 어디를 채우면 되는지 안내.

---

## 4. 자동 처리 항목

- 배송·통관 정보 (박유찬 / 인천 서구 가석로 30 / 통관구분 2 / 349-40-01104 등)
- 쇼핑몰 URL·브랜드 — 상품 URL에서 자동 판별 (1688 → www.1688.com / 1688)
- 수량 = 발주수량 × 개별수량 · 합계 = 단가 × 발주수량 · 묶음번호 = 1
- 트래킹번호·오더번호는 주문/배송 후 생기는 값이라 빈칸

---

## 5. 배대지 업로드

다운로드한 주문서를 배대지 대량등록에 업로드.
반려되는 칸이 있으면 메모해 전달 → 매핑을 맞춥니다.

---

## 한눈 요약

1. 장부: **개별수량 열 추가**(단가 그대로) + 빈 칸 채우기
2. 발주서·장부 업로드 → 변환 → 다운로드
3. 배대지에 업로드
