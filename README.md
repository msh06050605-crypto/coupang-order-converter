# 쿠팡 발주서 → 배대지 주문서 변환기

쿠팡(로켓배송) 발주서와 로켓장부를 업로드하면, 중국 배대지 대량주문서(.xlsx)를
자동 생성하는 브라우저 기반 단일 HTML 도구입니다. 서버 없이 브라우저 안에서
모든 처리가 이루어지며, 데이터는 외부로 전송되지 않습니다.

## 사용 방법
1. `index.html` 파일을 다운로드합니다.
2. 파일을 더블클릭해 크롬 등 브라우저로 엽니다.
3. 발주서(CSV)와 로켓장부(XLSX)를 업로드하면 변환 결과가 표시됩니다.
4. 엑셀 다운로드 버튼으로 배대지 주문서를 내려받습니다.

자세한 사용법은 [사용설명서.md](사용설명서.md) 참고.

## 특징
- 열 이름 자동 인식 (열 순서가 바뀌어도 동작)
- 같은 SKU 자동 합산 + 묶음번호 부여
- 합계 = 장부 단가(위안) × 발주수량
- 장부에 없는 품목은 미매칭 리포트로 분리
- 전 과정 브라우저 내 처리 (개인정보 외부 전송 없음)
